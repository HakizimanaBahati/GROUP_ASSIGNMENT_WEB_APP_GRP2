A web application project for group assignment, built with PHP.

## Structure

- **ASSIGNMENT1/** - First assignment with student management
- **ASSIGNMENT2/** - Second assignment with restaurant system (Question 1) and image gallery (Question 2)

## Assignment 1

Student management system with database operations.

## Assignment 2

- **Question 1** - Restaurant website with admin panel, menu, gallery, and order system
- **Question 2** - Image gallery application

## Setup

1. Clone the repository
2. Configure your database in the respective config files
3. Run the SQL setup scripts
4. Start your local server (e.g., XAMPP)

## Requirements

- PHP 7.4+
- MySQL
- Local web server
